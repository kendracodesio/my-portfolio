Weather App

Tools & Technologies:
VS Code, HTML, CSS, JavaScript, Axios, Bootstrap, API, GitHub

One of my favorite projects I've worked on to date was this weather web application that uses the Open Weather API. Using JavaScript with Axios, whenever the user enters a city name and clicks search, a request is made to the API server. If the city name is found, the weather data is retrieved from the third-party server and injected into the site. Additionally, the web app uses the Geolocation API to get the user's current position. Note: This is a optional feature that occurs only when location icon is clicked and user grants permission.

Typing Test App

Tools & Technologies:
VS Code, HTML, CSS, JavaScript, GitHub

This typing speed and accuracy test web application uses vanilla JavaScript to interact with the user. The test allows for three different options with a countdown timer that begins after the user clicks start. The app keeps track of the user's keystrokes using an event handler, allowing the user to see when they make a typing mistake in real time. Incorrect keystrokes are denoted in yellow and are underlined. Once the timer hits zero, the results screen is displayed showing the calculated words per minute and accuracy.

One of the first projects I created was this simple landing page for a coffee recipe e-book.
