Weather App

Tools & Technologies:
VS Code, HTML, CSS, JavaScript, Axios, Bootstrap, API, GitHub

One of my favorite projects I've worked on to date was this weather web app that uses the Open Weather API. Using JavaScript with Axios, whenever a city search is made, a request is made to the API server. If a match is found, the weather data is retrieved from the third-party server and injected into the site.

Typing Test App

Tools & Technologies:
VS Code, HTML, CSS, JavaScript, Github

This typing speed and accuracy test web application uses vanilla JavaScript to interact with the user. The test allows for three different options with a countdown timer that begins after the user clicks start. The app keeps track of the user's keystrokes using an event handler, allowing the user to see when they make a typing mistake in real time.
